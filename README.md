# AnalyseFichierPCAP

# Objectif 
récupération de données provenant de fichiers PCAP avec scapy. filtrage de données. 
Exploitation de ces données à l'aide de graphique pour en tirer des informations intérréssantes.

# Fonctionnalités 
  - récupérer et lire les données provenant de fichiers PCAP
  - outils d'analyse pour extraire des informations pertinentes ((calcul de délai, gigue, etc.)
  - Optimiser tout les processus pour traiter des fichiers volumineux

# Configuration recommandée
  - python
  - scapy
  - Panda
  - docker
  - Gonetem
  - Wireshark
  - GIT
